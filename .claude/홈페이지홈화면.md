### 홈화면

1. 상단배너 투명해서 잘 안보임, 모바일모드에서 뒷 배경 그대로 보이고 목록이 둥둥떠보임.
2. 삭제 영역

- 통계 보여주는 부분 삭제
- 최근 시공 사례 삭제

3. 히어로영역과 결과로 보여주는 기술력 사이에 들어가면 좋을내용 추천
4. 결과로 보여주는 기술력 영역 수정

- 현재: 작업별 사진 나열 + 설명 -> 수정: before/after 양 옆 혹은 위아래로 나열 + 설명
- 결과로 보여주는 기술력에 누른탭에 해당하는 시공 사례 카드를 아래 보여주도록(시공사례의 job_main_category로 필터)

### 시공사례 페이지 기능 추가

- 수정하기 버튼 클릭하면 관리자가 수정할 수 있도록
- 수정페이지에서도 사진 볼 수 있도록
- 수정페이지에서 사진삭제/사진삽입 가능하도록
