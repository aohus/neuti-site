# Track Specification: 메인 페이지 구현

## Overview
제공된 `Home.tsx` 구조를 바탕으로 '느티나무 협동조합 나무병원'의 홈페이지 메인 페이지를 구축합니다. 기존의 디자인 무드(자연 친화적, 전문성)를 유지하면서, 기술 스택인 Next.js와 Tailwind CSS에 맞춰 최적화된 방식으로 구현합니다.

## Functional Requirements
- **Hero Carousel (1섹션):**
    - 홍보 문구와 배경 이미지가 전환되는 캐러셀 슬라이더 구현 (Tailwind CSS 기반).
    - '조합 소개', '진단 의뢰' 바로가기 버튼 제공.
- **Mission Statement (2섹션):**
    - 회사의 핵심 가치를 담은 간결하고 임팩트 있는 배너 섹션.
- **Service Overview (3~4섹션):**
    - '조경/정원 사업'과 '나무병원(1종)' 분야의 주요 서비스 내용을 카드 및 이미지 형태로 소개.
- **Recent Updates & Projects (5섹션):**
    - 최신 공지사항 목록(최대 5개) 표시 및 전체보기 링크.
    - 주요 실적 프로젝트 갤러리(최대 4개) 표시 및 전체보기 링크.
- **Final CTA (6섹션):**
    - 하단 무료 견적 상담 유도 섹션.

## Non-Functional Requirements
- **기술 스택 준수:** Bootstrap 대신 Tailwind CSS를 사용하여 모든 UI를 새롭게 스타일링합니다.
- **디자인 일관성:** 기존 무드(Green/Brown 톤, 전문적이고 따뜻한 느낌)를 엄격히 유지합니다.
- **성능:** Next.js의 `Image` 컴포넌트와 비동기 데이터 패칭(API 연동)을 통해 최적의 속도를 구현합니다.
- **반응형:** 모바일에서도 모든 섹션이 자연스럽게 재배치되도록 구현합니다.

## Acceptance Criteria
- [ ] 제공된 코드의 모든 섹션이 Tailwind CSS로 재현되었는가?
- [ ] 캐러셀이 정상적으로 작동하고 이미지/텍스트가 올바르게 표시되는가?
- [ ] 최신 공지사항과 주요 실적이 백엔드 API와 연동되어 표시되는가?
- [ ] 모든 버튼과 링크가 해당 페이지로 정상 연결되는가?
- [ ] 모바일 환경에서 레이아웃이 깨지지 않고 가독성이 유지되는가?

## Out of Scope
- 관리자용 이미지 수정 기능 (`EditableImage`의 편집 기능 자체는 제외하고 렌더링에만 집중).
- 캐러셀의 매우 복잡한 애니메이션 효과.
