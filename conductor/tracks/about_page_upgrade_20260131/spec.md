# Track Specification: 회사소개 페이지 고도화

## Overview
기존의 회사소개 페이지를 전문가다운 신뢰감과 현대적인 감각이 조화된 페이지로 전면 업그레이드합니다. 잘못된 정보를 수정하고 연혁, 인증 및 특허, 지도 API 연동 등 신규 섹션을 추가하여 정보의 깊이를 더합니다.

## Functional Requirements
- **내용 수정 및 최적화:**
    - `assets/content_text/회사소개.md` 기반으로 회사 소개, 미션/비전, 핵심 역량을 정돈하여 반영.
- **연혁 (History) 섹션 추가:**
    - 별도의 `frontend/src/data/history.json` 파일을 사용하여 연도별 주요 성과 관리.
    - 수직 또는 수평 타임라인 형태의 시각적 컴포넌트 구현.
- **인증 및 특허 (Certifications) 섹션 추가:**
    - 아이콘과 리스트를 결합한 세련된 레이아웃으로 주요 인증 및 특허 내역 전시.
- **오시는 길 (Map) 섹션 추가:**
    - 카카오맵 또는 네이버맵 API를 연동한 인터랙티브 지도 삽입.
    - 본점 및 지점 주소, 연락처 정보 상세 제공.

## Non-Functional Requirements
- **기술 스택 준수:** Next.js (App Router), Tailwind CSS, Framer Motion (애니메이션).
- **디자인 일관성:** 메인 페이지와 동일한 '자연 친화적 & 전문적' 무드 유지.
- **유지보수성:** 연혁 데이터를 JSON으로 분리하여 업데이트 편의성 확보.

## Acceptance Criteria
- [ ] 회사소개 텍스트가 최신 본문에 맞게 정확히 표시되는가?
- [ ] 타임라인 섹션이 반응형으로 모바일에서도 가독성 있게 표시되는가?
- [ ] 주요 인증 내역이 전문적인 아이콘과 함께 깔끔하게 정돈되어 보이는가?
- [ ] 지도 API가 정상적으로 로드되며 정확한 위치를 가리키는가?
- [ ] 페이지 로딩 시 섹션들이 부드럽게 나타나는 애니메이션이 적용되었는가?

## Out of Scope
- 관리자 페이지에서의 실시간 데이터 수정 기능 (현재는 JSON 파일 기반).
- 다국어(영문 등) 버전 지원.
