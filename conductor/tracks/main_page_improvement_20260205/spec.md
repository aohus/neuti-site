# Specification: 메인 페이지 UI 개선 (홍보 전략 강화)

## 1. 개요 (Overview)
- **목적:** 3년 차 신생 기업의 한계를 넘어서는 '데이터 기반 신뢰'를 구축하고, 아파트 및 공공기관의 대형 작업(방제, 전정) 수주를 위한 전략적 관문 마련.
- **핵심 메시지:** "데이터로 증명하는 나무의사의 정밀한 관리, 공공기관이 선택한 검증된 기술력"

## 2. 주요 기능 및 요구사항 (Functional Requirements)

### A. 데이터 기반 신뢰 대시보드 (Trust by Numbers)
- `공사실적.md` 데이터를 분석하여 자동/수동으로 집계된 핵심 지표 노출.
  - *예시:* "3년간 공공기관 협력 150건+", "관리 수목 10,000주 이상", "성남시 관내 주요 공원 5개소 정밀 방제 수행"
  - 애니메이션 효과가 있는 카운터 UI 적용.

### B. 타겟 고객별 진입로 (Targeted CTA Zones)
- 메인 중단에 고객군별 맞춤형 카드 배치:
  - **[아파트/기업용]** "체계적인 수목 관리로 자산 가치를 높이세요" -> 관리 제안서 보기.
  - **[공공기관용]** "나무의사의 과학적 진단과 정밀 시공" -> 주요 공공 실적 포트폴리오.
- 각 카드 클릭 시 해당 고객군이 관심 있을 실적 필터링 페이지 또는 전용 안내로 연결.

### C. 전문가 프로세스 & 성과 입증 (Professional Proof)
- 단순 갤러리를 넘어선 '솔루션 중심' 섹션:
  - '단대/양지공원' 사례와 같이 **[진단(예찰) -> 처방 -> 방제 -> 결과보고]** 프로세스를 시각화.
  - 전/후 비교(Before & After) 이미지 슬라이더 및 '방제 효과 분석' 지표 노출.

### D. 전략적 레이아웃 재구성 (Narrative Flow)
1. **Hero Section:** 나무의사의 전문성을 담은 카피와 핵심 신뢰 지표.
2. **Client Banner:** (기존 유지) 공공기관 로고 롤링 배너.
3. **Data Dashboard:** (신설) 숫자로 보는 신뢰도.
4. **Target Services:** (강화) 방제, 전정 등 주력 대형 작업 안내.
5. **Success Story:** (신설) 효과 중심의 베스트 사례.
6. **Target CTA:** (신설) 아파트/공공기관 전용 진입로.

## 3. 비기능 요구사항 (Non-functional Requirements)
- **반응형 최적화:** 현장 담당자(관리소장 등)가 모바일로 확인 시에도 신뢰감을 주는 레이아웃.
- **이미지 최적화:** 고해상도 작업 사진을 사용하되 로딩 속도 저하 방지.

## 4. 수락 기준 (Acceptance Criteria)
- 홈 화면 진입 시 3초 이내에 회사의 전문성과 실적 규모를 직관적으로 파악할 수 있는가?
- 아파트 관리자 및 공공기관 담당자가 자신의 필요에 맞는 정보를 쉽게 찾을 수 있는가?
- 방제/전정 작업의 '과학적 프로세스'가 시각적으로 충분히 전달되는가?
